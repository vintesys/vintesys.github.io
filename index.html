Tecnologias

Estamos sempre buscando as melhores ferramentas do mercado, além de buscar conhecimento para entregar o melhor produto para nossos clientes.

Flutter

Flutter é um framework de desenvolvimento de aplicativos móveis criado pelo Google. Ele permite que os desenvolvedores criem aplicativos nativos para iOS e Android a partir de um único código base, usando a linguagem de programação Dart.

Firebase

Firebase é uma plataforma de desenvolvimento de aplicativos móveis e web que é oferecida pelo Google. Ele fornece uma ampla gama de serviços de backend na nuvem que ajudam os desenvolvedores a criar aplicativos de alta qualidade com rapidez e eficiência.

Heroku

Heroku é uma plataforma de nuvem que permite aos desenvolvedores hospedar, executar e escalar aplicativos web em diferentes linguagens de programação.

Java

Java é uma linguagem de programação orientada a objetos, usada para desenvolver aplicativos de software para uma ampla gama de plataformas, incluindo desktops, servidores, dispositivos móveis e a web.
